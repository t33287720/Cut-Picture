介紹
=
定時裁切流水號圖片(從0開始)為指定大小

裁切位置需自行調整

適用於定點拍攝擷取重點變化

裁切時按任意鍵則結束運行

方法一
-
執行單一程式碼(Find_and_cut)

顯示0.jpg 左右鍵選擇需裁切之位置

以output 確認裁切範圍

按Enter後開始定時裁切並另存檔案(定時10秒)

方法二
-
執行兩程式碼-Find_location尋找位置

以output 確認裁切範圍

按Enter關閉視窗

後執行兩程式碼-Cut_and_store

將Find_location最後之輸出位置貼上

輸入後開始定時裁切並另存檔案(定時10秒)
